Estrutura do Banco de Dados

Tabelas Principais
Tabela Registros Descrição
adm~10Administradores do sistema
clientes~500Clientes cadastrados
requisicoes~200Requisições de crédito
contador_visitas~10kControle de visitas
depoimentos~50Depoimentos de clientes
logs_auditoria~1kLogs de auditoria
reset_senha~20Solicitações de reset
previsoes_pico~700Previsões ML
Relações e Funcionalidades
  • adm → logs_auditoria: Rastreamento de ações
  • clientes → requisicoes: FK cliente_id
  • requisicoes → clientes: JOIN para dados completos
  • contador_visitas: IPs únicos, timestamps
  • depoimentos: Status (pendente/aprovado)
  • reset_senha: Tokens únicos, controle de status
Script SQL: sql/esquema_completo_confinter.sql
